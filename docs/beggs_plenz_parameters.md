## 0. 適用済みの較正 (2026-07-20)

`beggs_plenz.yaml` / `beggs_plenz_n1000.yaml` は物理単位版ニューロン
`akita_escape_lif_physical` を使う。それに合わせて `g_max` を §7.3 の較正値へ更新した。

| | 更新前 | 更新後 | 根拠 |
|---|---|---|---|
| `e-stdp_beggs` の `g_max` | 4.0 | **0.0086 µS** | EPSP 0.7 mV (§7.3) |
| `i-stdp_beggs` の `g_max` | 4.0 | **0.0429 µS** | IPSP 2.0 mV @ −55 mV、g_I/g_E = 5.0 (§7.3) |
| `beggs_plenz_n1000.yaml` の neuron | `akita_escape_lif` | `akita_escape_lif_physical` | 単位系を揃えるため |

`i-stdp_beggs_gi{2,3,4,6,8,10}` を追加してある(末尾 = g_I/g_E)。臨界点探索の掃引用。
gi≤4 は暴走域(~92 Hz)なので実効探索範囲は 5〜10。

**IPSP 較正の注意 (2026-07-21 確認)**: g_I の較正は**抑制シナプスの減衰 tau_syn=4 ms**
(ExpCond inhibitory。興奮の 2 ms ではない)を使うこと。誤って 2 ms で計算すると同じ 0.0429 µS が
出す IPSP が 2.0→1.15 mV に見え、g_I/g_E を 5.0→8.8 と過大評価する。§7.3 の表は 4 ms で正しい。

⚠️ **更新前の 4.0 は無次元版の値で、物理単位版ニューロンと組み合わさると
`4.0 × R_m = 400` 相当になっていた。**ただし、当初観測された
「N=40000 の発火率が異常に高い」現象は**物理単位版を導入する前**のものなので、
この不整合はその原因ではない。単位系が混ざっていた期間の実行結果のみが無効。

## 1. 単位系

物理単位版ニューロン `akita_escape_lif_physical` を使うので、以下で閉じる。

| 量 | 単位 | 備考 |
|---|---|---|
| 膜電位 | mV | |
| コンダクタンス `g`, `g_max` | **µS** | |
| 膜抵抗 `R_m` | **MΩ** | |
| 電流 `Isyn`, `Iext` | **nA** | µS × mV = nA |
| 導出 `g_rest = 1/R_m` | µS | |
| 導出 `C_m = tau_m/R_m` | nF | ms / MΩ = nF |

積分式: `V += ((V_rest - V) + R_m·(Isyn + Iext)) · dt/tau_m`

無次元版との対応: **`g_max_無次元 = g_max_物理[µS] × R_m[MΩ]`**
(GeNN 実機でビット一致することを `test/core/test_genn_reproducibility.py` で検証済み ✅)

## 2. ネットワーク規模と空間配置

| パラメータ | 値 | 根拠 | 状態 |
|---|---|---|---|
| 空間 | 1500 × 1500 µm 2D ランダム | Beggs & Plenz 2003 の 8×8 MEA、電極間 200 µm → 有効記録野 1400 µm。それを覆う正方形 | ✅ MEA 仕様は確認 |

**解釈**: 標本は器官型培養スライスだが、パラメータは急性スライス/in vivo の

この点は結果の解釈時に留保が必要。

### 縮小ラダーについて 🔶

`random_2d_n1000` 等は「面積密度 0.017778 neurons/µm² を保つ」設計になっているが、
**実測では出次数が収束しない**(N=1000→190, 4000→530, 10000→834, 40000→1202)。
σ_ee = 300 µm が箱の一辺に対して大きすぎ、有限サイズ効果が支配的なため。
**縮小版は smoke test 専用**であり、物理のプレビューには使えない。

## 4. 結合確率 (`connections.yaml: beggs_plenz`)

`P(d) = p0 · exp(-d²/2σ²)` の種別別パラメータ。

| 経路 | σ [µm] | p0 | 根拠 | 状態 |
|---|---|---|---|---|
| E→E | 300 | 0.20 | ラット皮質 L2/3 水平軸索の拡がり ~500 µm (Hellwig 2000) | ⚠️ 論文は結合確率 50–80%(隣接)〜0–15%(500 µm)を報告。σ への変換は解釈 |
| E→I | 200 | 0.30 | 抑制性細胞へはより局所的 (Fino & Yuste 2011) | ⚠️ |
| I→E | 175 | 0.25 | 皮質抑制は空間的に局所 (Fino & Yuste 2011) | ⚠️ |
| I→I | 125 | 0.20 | 最も局所 (Fino & Yuste 2011) | ⚠️ |
| 自己結合 | 禁止 | | Bekkers & Stevens 1991 | ⚠️ |

### 実測された fan-in ✅

実際にネットワークを構築して数えた結果:

| config | N | E→E fan-in (K_E) | I→E fan-in (K_I) | K_E/K_I | 総シナプス数 |
|---|---|---|---|---|---|
| beggs_plenz | 40000 | **1134.4** | 140.1 | 8.09 | 48,090,437 |
| beggs_plenz_n1000 | 1000 | 144.5 | 37.9 | 3.81 | 189,614 |

**K_E = 1134 は生物学的に妥当**である。McCormick et al. 2003 のレビューによれば
皮質錐体細胞は約 10,000 のシナプス入力を受け、うち約 75%(≈7500)が興奮性。
Hellwig 2000 の再構成では 1 結合あたりの接触点数は多くの場合 1、近接ペアで平均 3 程度。
したがって distinct な前興奮性ニューロン数は **2500〜7500** と見積もられ、
K_E = 1134 はこの範囲のやや下限側。有限の培養標本であることを考えれば妥当。

⚠️ **ただし K_E/K_I = 8.09 は N 依存**である(N=1000 では 3.81)。
σ_ie(175 µm) が σ_ee(300 µm) より局所的なため、箱が大きくなると
興奮性入力だけが遠方から集まり続ける。これは σ の設定の帰結であって
意図した設計ではない可能性が高い。**要検討事項**。

## 5. 伝導遅延 (`delays.yaml: beggs_plenz`)

`delay(d) = d / velocity + synaptic_delay`

| パラメータ | 値 | 根拠 | 状態 |
|---|---|---|---|
| `velocity` | 500 µm/ms | 無髄軸索伝導速度 (Swadlow 1994) | ⚠️ + 🔶 下記参照 |
| `synaptic_delay` | 1.0 ms | 最小シナプス遅延 (Sabatini & Regehr 1996) | ⚠️ |
| `max_delay` | 25.0 ms | 🔶 **実装都合** | ✅ 下記 |

🔶 **`max_delay = 25.0` は生物学的な値ではない。**
NetworkBuilder が遅延を uint8 ステップで持つため、`255 × dt = 25.5 ms` を超えると
**無言で折り返す**。そのキャップ。同じ理由で `velocity = 500` も、
実測中央値としてよりは「1500 µm の対角線で折り返さない」ための下限として効いている。
恒久対策(uint16 化など)は未実施。

## 6. 初期重み (`weights.yaml: beggs_plenz`)

対数正規分布を [0, 1] でクリップ。

| パラメータ | 値 | 根拠 | 状態 |
|---|---|---|---|
| `mu_ln` | −0.857 | `= ln(0.7) − σ²/2`、中央値 0.7 相当 | ✅ 算術は整合 |
| `sigma_ln` | 1.0 | 分布の広がり | ⚠️ |
| `w_min` / `w_max` | 0.0 / 1.0 | custom_Akita のハードバウンドと一致 | ✅ |

**根拠**: シナプス強度が対数正規分布に従うことは Song et al. 2005 (PLoS Biol) が
L5 錐体ペア数百組の記録から明示的に報告している ✅
("the distribution of synaptic connection strength ... can be fitted by a lognormal distribution")。

⚠️ **注意**: `w` は無次元の [0,1] スケール係数であり、実効コンダクタンスは
`x_release · w · g_max` である。したがって `mu_ln` に埋め込まれた "0.7" は
**mV 単位の EPSP ではなく、w の中央値**。較正では `w = 0.5` を代表値として使っている。
ここは表記が紛らわしいので、`mu_ln` のコメントにある「平均 ~0.7 EPSP」は誤解を招く。

## 7. シナプス伝達と可塑性

### 7.1 コンダクタンス時定数 (`synapses.yaml: ExpCond`)

| | τ [ms] | E [mV] | 状態 |
|---|---|---|---|
| 興奮性 (AMPA) | 2.0 | 0.0 | ✅ 標準値 |
| 抑制性 (GABA_A) | 4.0 | −80.0 | ✅ 標準値 |

### 7.2 短期抑圧 (Tsodyks-Markram)

| パラメータ | 値 | 状態 |
|---|---|---|
| `U` (初回放出率) | 0.4 | ⚠️ |
| `tau_rec` (回復時定数) | 150 ms | ⚠️ |

**これは無視できない効果である** ✅。定常発火率 r に対する平均資源は
`<x> = 1/(1 + U·tau_rec·r)` で、時間平均放出量は

```
U·<x>·r = U·r / (1 + U·tau_rec·r)   →   1/tau_rec = 6.67 /s  (r → ∞)
```

と**飽和する**。すなわち短期抑圧自体が暴走に対する負のフィードバックとして働く。

| r [Hz] | `<x>` | x=1 と仮定した場合の過大評価 |
|---|---|---|
| 1 | 0.943 | 1.06× |
| 5 | 0.769 | 1.30× |
| 20 | 0.455 | 2.20× |
| 50 | 0.250 | **4.00×** |

⚠️ 近似: 多接触結合(実際は 1〜3 接触)を単一放出部位として扱っているため、
抑圧の効き方は実際より階段的になる。平均振幅には影響しない。

### 7.3 `g_max` — 単発 PSP からの較正 ✅

**手続き**: 文献が測定したのと同じ条件をモデル上で再現し、PSP 振幅が一致する
`g_max` を数値的に解く。E/I 比を自由パラメータとして合わせ込むことは**しない**。

#### 興奮性 — 確定できる

| | 値 |
|---|---|
| 参照 | **EPSP 0.7 ± 0.6 mV**、静止電位付近、L2/3 錐体ペア (Feldmeyer et al. 2002 J Physiol, PMC2343144) |
| 解釈 | unitary EPSP は「1 個の前ニューロンの 1 スパイクが後細胞に生む脱分極の**総和**」。接触点が複数あればその合計。モデルのシナプス 1 本も同じ量(前ニューロン i → 後ニューロン j の 1 結合)を表すので、**接続レベルで対応が取れている**。K_E が接触点数ではなく前ニューロン数を数えていることが、この対応の前提 ✅ |
| 結果 | **`g_max_E` = 0.0086 µS (無次元 0.863)** |
| 精度 | 高い。駆動力が `0 − (−74) = 74 mV` と大きく正確に分かるため |

#### 抑制性 — 帯にしかならない ⚠️

| | 値 |
|---|---|
| 参照 | **IPSP 0.2 – 3.5 mV**、静止電位 **−55〜−60 mV**、FS 介在細胞 → 錐体細胞、逆転電位 **−70〜−82 mV** (Thomson, West, Hahn & Deuchars 1996 J Physiol, PMC1160826) |
| なぜ違う電位で測るのか | GABA_A 逆転電位が静止電位のすぐ近くにあるため、静止電位では駆動力がほぼゼロで IPSP を測定できない。逆に EPSP は脱分極させると発火してしまう。**測定手技の都合であって生理学的な差ではない**。ゆえに比較すべき不変量は振幅ではなくコンダクタンス |
| 問題 | 駆動力 `V_hold − E_Cl` が **15〜27 mV** と小さく、かつ `E_Cl` 自体が記録条件(ピペット内液の Cl⁻ 濃度)依存の幅を持つ。分子(IPSP)も 0.2〜3.5 mV と広い |

`g_max_I` [µS](括弧内は `g_I/g_E`、g_E=0.0086)。
EPSP 較正と同一の過渡ピーク積分(R_m=100 MΩ, tau_m=30 ms, **抑制 tau_syn=4 ms**, U·W=0.2, E_inh=−80):

| IPSP \ 保持電位 | −50 mV | −55 mV | −60 mV |
|---|---|---|---|
| 0.2 mV | 0.0034 (0.4) | 0.0041 (0.5) | 0.0051 (0.6) |
| 1.0 mV | 0.0174 (2.0) | 0.0209 (2.4) | 0.0263 (3.1) |
| **2.0 mV** | 0.0354 (4.1) | **0.0429 (5.0)** | 0.0543 (6.3) |
| 3.5 mV | 0.0640 (7.4) | 0.0781 (9.1) | 0.0999 (11.6) |

**採用: 2.0 mV @ −55 mV → g_I = 0.0429 µS (g_I/g_E = 5.0)**(ユーザ確認済 2026-07-21)。

**結論: 文献の IPSP 幅 (0.2–3.5 mV) と保持電位幅 (−50〜−60) で `g_I/g_E` は 0.4〜11 の帯。
生理的に妥当な 1–2 mV @ −55 に絞れば 2.4〜5.0。単発コンダクタンス直接測定
(Gupta 2000 / Xiang 2002、いずれも全文有料で値未抽出)があれば駆動力不定性を回避できる。**

なお、体細胞周囲に投射する FS 介在細胞と樹状突起スパインに投射する興奮性入力では
樹状突起フィルタリングの受け方が異なるが、**点ニューロンモデルでは発火に効くのは
体細胞での効果**なので、体細胞記録値をそのまま使うのは整合的である ✅。

## 8. 平均場による動作点の確認 ✅

短期抑圧を含めた定常膜電位(閾値 −54 mV)。較正値 `g_E = 0.863`,
`g_I = 3.54`(g_I/g_E = 4.1、帯のほぼ中央)を使った場合:

| | 0.5 Hz | 1 Hz | 2 Hz | 5 Hz |
|---|---|---|---|---|
| N=1000 | −72.6 | −71.5 | −69.8 | −66.7 |
| **N=40000** | −64.7 | **−59.6** | −54.3 | −48.6 |

N=40000 に**閾値下の低発火解が存在する**。1 Hz で閾値の 5.6 mV 下、2 Hz でちょうど閾値
という位置で、臨界近傍として妥当な動作点。

⚠️ **平均場の適用範囲**: この計算は各前ニューロンが独立に率 r で発火する前提の
**時間平均**である。アバランシェは定義上**同期**現象なので、イベント中の瞬時入力は
この値を大きく上回る。平均場は「静穏時の動作点が閾値下にあるか」の判定にのみ使えて、
臨界性そのものの予測には使えない。

## 9. 意図的に採用しなかった選択肢

| 案 | 却下理由 |

## 10. 未解決事項

1. **§0 の単位系不整合の修正** — 最優先。
2. **`g_max_I` の決定** — 文献の帯 `g_I/g_E ∈ [2, 10]` を掃引し、
   branching parameter σ ≈ 1 になる点を探す(測定不定性の範囲内の掃引であり、

3. **`K_E/K_I` の N 依存性** — σ_ie < σ_ee に由来する構造的な問題。設計意図の確認が必要。
4. **⚠️ 印の一次資料確認** — 特に神経細胞密度、Fino & Yuste 2011 からの σ の導出、
   `U` / `tau_rec` の出典。
5. **遅延の uint8 折り返し** — `max_delay` によるキャップは暫定回避。
6. **アバランシェ発生率の較正** — 現状 137/s に対し文献値は ~1.3/s。
   ただし論文の電極数・LFP ベースの検出とシミュレーションのニューロンベース検出は
   別物なので、絶対数の一致は目標にしない方針。
