# Akita SoC-SNN 再現 技術メモ

対象論文: N. Ikeda, D. Akita, H. Takahashi, *"Noise and spike-time-dependent plasticity drive
self-organized criticality in spiking neural network: towards neuromorphic computing,"*
Appl. Phys. Lett. 123, 023701 (2023)（およびSupplementary, 以下 S 式番号）。
祖先モデル: N. Stepp, D. Plenz, N. Srinivasa, PLoS Comput. Biol. 11, e1004043 (2015)。

目的: 論文 **Fig 2（自己組織化臨界の発達過程）** を同一条件で再現する。
結論を先に: **公開式 (S1–S17)・Table S1 に忠実に実装しても、Fig 2 の「安定した臨界の発達」を再現できない。**
再現できるのは 6h 前後の一時的な臨界スナップショット（アバランシェ分布・ラスター・LLR≈2万）のみで、
指標の時間発展（exc/inh 発火率, LLR, ΔCr, BI, D）は論文の安定臨界を維持できない。

## 1. 実装と忠実性の検証（すべて一致を確認済み）

- 基盤: PyGeNN 5.4 / GeNN, **double 精度**, escape-noise conductance-based LIF + STP + E/I-STDP。
- 公開式・パラメータを実装し、**生成 C++ コードまで照合**して一致を確認:
  - 膜方程式 S1、コンダクタンス更新 **S9: `g_exc += U·x·w·g_max`**(fan-in正規化なし)、減衰 S7/S8 (τAMPA=2, τGABA=4ms)、
    escape noise S5/S6 (C≈0.00594, rest 0.4Hz)、STP S11/S12 (U=0.4, τrec=150)、E-STDP S16、I-STDP S17。
  - Table S1 全パラメータ一致 (N=100=80E/20I, all-to-all, 初期重み0, 遅延 EE=1.5/他0.8ms, A_E=A_I=0.02,
    τE=20/τI1=10/τI2=20, βE=1.0/βI=1.15, g_max_exc=g_max_inh=4.0)。
- **STDP 数式の独立検証**: `test/core/STDPtest/test_multi_spike.py` にて、全対全 trace (delay_corrected) の
  E/I-STDP を「全ペア和の解析リファレンス」と照合 → 全シナリオ **|diff| ~1e-16** で一致。
- **数値の健全性**:
  - 時間精度 double（72h で t≈2.6e8 ms, 精度~6e-8 ms ≪ STDP判定窓0.05ms）。tの肥大化は無害。
  - **dt 不変性**: dt=0.1/0.05/0.025 で発達はほぼ同一（dt=0.025で~10%のみ）→ 前進Euler離散化は2x速度の原因ではない。
- **解析指標の論文準拠性**:
  - アバランシェ閾値 = 平均スパイク間隔（S）, BI (S25), D=降順サイズの最大ギャップ (S26), LLR(離散MLE) … 一致。
  - **ΔCr (S21–S24) は 2026-07-05 に重大バグを発見・修正**（下記 §8）。旧実装は偽の強い劣臨界値を出していた。
  - **LLR**: 当初は簡易近似だったため、Clauset(2009)/Yada(2017) 流の **離散打ち切り最尤フィット([1,100], power-law vs exponential)**
    に修正済み（`src/utils/akita_soc.py`）。修正後、h6 の LLR≈18900 が論文の~20000 とほぼ一致するようになった。

## 3. 論文の発達系列（重要な前提）

論文 Fig 2 の発達は **subcritical → 6h=超臨界(supercritical) → 72h=臨界(critical)** の二段階：
- **~6h**: アバランシェ分布が **二峰性(bimodal, D高)= 超臨界** に到達（同期バースト、branching σ>1）。
- **6h→72h**: 可塑性による自己組織化で **超臨界から臨界へ緩やかに降下**（D低下・power-law化・σ→1）。
- **~72h**: **臨界(power-law, ΔCr≈0)** に到達し安定。

注意: 6h は臨界**ではなく超臨界**。SoCの本質は「一度σ>1にオーバーシュートし、可塑性がσ=1へ自己調整して**留まる**」点。

### 定量比較（βE=1.0/βI=1.15, A=0.0125, seed2; 我々は0-12hのみ）
| 指標 | 論文(6h超臨界→72h臨界) | 我々 h6 | 我々 h12 |
|---|---|---|---|
| exc rate | 2.0 → 1.5/0.5 二分化 | 5.1 | 0.87 |
| inh rate | 2.5 → 3.0 | 7.7 | 4.0 |
| LLR(修正版) | 高値で推移 | ~18900 | 4894（低下） |
| D (bimodality) | 6h高(~300, 二峰性)→72hで低(臨界) | 12〜326と不安定 | 28 |
| ΔCr | 6h正(超臨界)→72h≈0(臨界) | +0.03 | **-0.56(劣臨界)** |

- 我々は超臨界の**一時的オーバーシュートは作る**（Dの一時スパイク）が、**6h→72hの「臨界への自己組織化」を再現できない**。
- A=0.0125 では h6 以降、抑制が過剰発達(IE→0.85)し興奮崩壊 → **臨界を通り越して劣臨界へドリフト**（ΔCr→-0.56）。
- A=0.02 では逆に抑制崩壊 → 超臨界のまま暴走(123Hz)。**いずれも σ=1(臨界) で止まれない**。

## 4. 核心的な未再現点（現象論）

### branching 比 σ の定義
neuronal avalanche における **branching 比 σ** は「**1個のスパイク（親）が次世代に直接引き起こすスパイク数の期待値**」:

    σ = E[ 次世代のスパイク数 / 現世代のスパイク数 ]

- **解析的推定 (Beggs & Plenz 2003)**: アバランシェ内を時間ビン Δt で区切り、連続ビンのスパイク数を n_1, n_2, … としたとき
  σ = ⟨ n_{t+1} / n_t ⟩（最初のビンを「親世代」とみなす）。等価に「祖先1スパイクあたりの子孫スパイク数の平均」。
- **物理的意味（分岐過程としての解釈）**: σ = (下流ニューロン数) × P(下流ニューロンがその1スパイク由来の入力で発火する)。
  本モデルでは、1個の pre 発火が post j に与える EPSP は ΔV_j ∝ U·x·w_ij·g_max·(E−V)、
  escape noise が発火確率を ∝ exp(ΔV_j / b) で増やす。よって
  σ ≈ Σ_j [この増分発火確率] となり、**重み w・STP資源 x・g_max・ニューロンgain(b)・抑制(IE)** に依存する実効量。

- **σ<1（劣臨界）**: アバランシェは指数的に消滅（小サイズに偏り、LLR低・ΔCr<0）。
- **σ=1（臨界）**: アバランシェサイズが **power-law（スケールフリー）**（LLR高・ΔCr≈0・D小）。
- **σ>1（超臨界）**: アバランシェが発散し系全体に波及 → 大規模イベントが分離して **二峰性分布**（D の大ギャップ、ΔCr>0）。

SoC の自己調整とは、STP（x の枯渇）・E-STDP（w）・I-STDP（抑制）を通じて **実効的な σ を 1 へ引き込む** 制御に相当する。
本メモの指標との対応: **ΔCr≈0 & LLR高 ⇔ σ≈1（臨界）**、**D の大ギャップ（二峰性）⇔ σ>1（超臨界）**、
**LLR低 & ΔCr<0（小アバランシェのみ）⇔ σ<1（劣臨界）**。

### SoC の制御ループ（論文が依拠する機構）
SoC は「**branching 比 σ を可塑性が σ=1 へ自己調整し、臨界を安定な吸引点(stable fixed point)として保持する制御ループ**」:
- σ>1(超臨界): 活動↑ → STP資源枯渇 ＋ I-STDP が抑制増強 → σ↓
- σ<1(劣臨界): 活動↓ → 資源回復 ＋ E-STDP増強 ＋ I-STDP抑制解除 → σ↑
- → σ=1(臨界) が安定吸引点。発達は「成長 → σ>1へオーバーシュート(6h, 超臨界) → 緩降下 → σ=1で停止(72h, 臨界)」。

### 我々の網: 超臨界は作れるが「臨界で止まれない」
- 超臨界へのオーバーシュートは起きる（D の一時スパイク・二峰性）。
- しかし **σ=1 で停止できず、安定臨界吸引点が無い**:
  - A=0.02: 超臨界遷移時に I-STDP が抑制を増強せず **崩壊**（正味抑圧的カーネルが高活動で逆向きに働く）→ σ↑ で暴走。**制御ループが高活動側で符号を誤る**。
  - A=0.0125: 降下するが **σ=1 を通り越して σ<1(劣臨界)へ**。**固定点が臨界より下にずれている**（過抑制）／降下が不足制動。
- → 足りないのは個別パラメータでなく、**「臨界(σ=1)を安定吸引点に持つ恒常性制御」そのもの**。

### 臨界の固定点を決める要素（どれかがずれると留まれない）
1. **I-STDP の恒常性セットポイント**: 目標発火率が σ=1 に一致する必要。I-STDPカーネル(βI,τ)×ニューロンf-I gain×発火相関で決まり、不一致で固定点が臨界からずれる（我々は劣臨界側＝目標σ<1）。
2. **STP(短期抑圧)のフィードバック強度**: Stepp/Levina系では STP が σ を1へ引き戻す主アクチュエータ。資源x→効力→σ の結合が論文と少しでも違うと到達点がずれる。
3. **ニューロンの f-I gain (escape noise, b=4mV)**: σ の感度を決める。急峻な gain は σ=1 を「狭い的」にし、わずかなE-I不均衡で σ が大きく振れ、オーバーシュート/不安定化。
4. **時間スケール分離**: 可塑性が活動より十分遅く振動せず収束する必要（A=0.02は速すぎ発散、A=0.0125でも収束先がずれる）。

### 高時間分解能(0.25h刻み)解析で見た「分岐の正体」(legacy vs delay_corrected, A=0.02, seed1)

遷移を 0.25h 刻みで記録し、発火の同期性も spike から実測した。両者とも **同じ準臨界バースト点に到達した後、運命が分かれる**。

| | legacy(発火時刻基準) | delay_corrected(到着時刻基準) |
|---|---|---|
| 分岐点 | h2.5: バースト＋準臨界(bimod222, ΔCr-0.09, EE0.17/IE0.19) | h3: バースト＋準臨界(bimod16, ΔCr+0.02, EE0.29/IE0.70) |
| 直後に暴走する興奮性経路 | **EE(exc→exc)**: 0.17→0.71 | **EI(exc→inh)**: 0.38→0.81 (h3→h4) |
| バースト(spike実測) | h2.75で**連続tonic化**(無音0.00,CV0.15) | h2.5-3.25で**離散バースト維持**(無音0.6-0.86,CV2.4-4.3) |
| 抑制の運命 | IE: 0.19→**0(崩壊)** | IE: 育つ(→0.92)が**EI駆動でinh過剰**(inh/exc→4.5) |
| 帰結 | **σ>1(超臨界runaway, 112Hz)** | **σ<1(過抑制, 劣臨界, ΔCr-0.6)** |

**重要な訂正(機構の精緻化)**: legacy の IE→0 は「バースト中に I-STDP が抑制を削る」のではない。
- h2.5(同期バースト)では I-STDP は密な相関ペア(|dt|<11ms)を**増強**し、IEは実際に0.19まで育つ。
- その後 **バーストが連続tonic発火へ融合(同期消失, CV0.15)** して初めて、I-STDPが低相関/async regimeに入り **正味depressionでIE崩壊**。
- つまり **async積分による抑制は「同期消失後の連続状態」でのみ妥当**で、同期バースト状態には当てはまらない。IE崩壊は**同期消失のロックイン(結果)**。

### 統一仮説: βE=1.0 で**興奮性結合(EE, EI)に復元力が無い**ことが両failure modeの共通根

- βE=1.0 → E-STDP正味積分 ∫F_E = A_E·τ_E·(1−βE) = **0** → **EEもEIも上限への復元力を持たず**、活動相関＋境界ドリフトで正のフィードバック的に暴走しやすい。
- 遷移点(σ≈1)は **ナイフエッジ**。どちらの興奮性経路が先に暴走するかで運命が決まる:
  - **EE(リカレント興奮)が先に暴走 → バースト融合 → σ>1**（legacyで顕著。発火時刻基準でわずかに高温＝融合が速い）
  - **EI(exc→inh駆動)が先に暴走 → inh過剰駆動 → σ<1**（dcで顕著。バースト離散維持だがEIが復元力なく飽和→過抑制）
- **どちらもσ=1で止まれない**＝興奮性結合を臨界平衡で止める復元力/恒常化が無い。seed次第でdcもEE融合側(σ>1)へ倒れる（暴走seedの正体）。
- これは§4冒頭の「臨界の安定吸引点が無い」を、**具体的な暴走経路(EE vs EI)とナイフエッジ双安定性**として特定したもの。

### 検証実験の結果: 根本原因は escape noise の gain (b)。興奮性の復元力(βE)では不発

統一仮説の2介入を並列検証（各 seed 2-6, A=0.02, g_max=4.0, 12h）:

| 条件 | 変更(他は論文準拠) | EI(h12) | IE | inh/exc | ΔCr | 結果 |
|---|---|---|---|---|---|---|
| baseline | βE=1.0, b=4 | 0.95飽和 | 0.92 | 4.4 | -0.59 | 過抑制 σ<1 |
| b105 | **βE=1.05**(興奮性に復元力) | 0.96飽和 | 0.86 | 4.5 | -0.6〜-0.7 | **過抑制 σ<1（不発）** |
| b8 | **b=8mV**(gainを浅く, 2倍速を補償) | **0.50** | **0.10** | **1.3** | **-0.19〜+0.18** | **near-critical σ≈1（成功, 全5seed）** |

- **b8で初めて σ=1 を安定保持**: ΔCr が0を挟んで振動、LLR正、**抑制が飽和しない(IE0.10, II0.12)**、E-I均衡(inh/exc 1.3)、moderate weights。遷移も h3→h8 と遅くなり論文の~6hに接近。
- **b105は不発**: 興奮性への弱い復元力ではEI飽和・過抑制σ<1が止まらない → 「興奮性の復元力欠如」は主因ではない。
- **結論: 根本原因は escape noise の gain が急峻すぎたこと(b=4)**。b小 → σ=1が「狭い的」→ 2倍速い重み成長を恒常性が追従できず暴走/過抑制へ外れる。**bを2倍(8mV)にして dσ/dw を半減し2倍速を補償**するとσ=1が安定吸引点になる。「2倍速い重み成長」が σ=1不安定として顕現し、gainで補償できることを示す。
- **要確認**: b8のIEが h9→12 で 0.08→0.10 と緩上昇中 → 72hまで保持するか長時間ランで検証要。定量一致(LLR~1600 vs 論文2万, inh率1.5 vs 3)は不完全で b の微調整余地あり(b∈4〜8)。

## 5. 結論

- 実装は公開式 (S1–S17)・Table S1 に忠実（生成コード・単体テスト・dt不変性・double精度で確認）。
- 論文の網は **σ=1(臨界) を安定吸引点に持つ自己調整臨界系**で、発達は「超臨界(6h)へオーバーシュート → 緩やかに臨界(72h)へ降下して留まる」。
- 我々は公開式に忠実だが、**超臨界は作れても臨界の安定吸引点が無い**ため、制御が **発散(暴走, A=0.02) か行き過ぎ(劣臨界, A=0.0125)** になり、**6h→72hの臨界への自己組織化を再現できない**。
- 不足しているのは単一パラメータでなく、**STP ＋ I-STDP恒常性 ＋ ニューロンgain の精密な釣り合いが臨界(σ=1)に一致すること**＝SoC機構の核そのもので、これが未公開の実装詳細に依存していると考えられる。
- パラメータ・STDP変種の総当たり（A, g_max, βE/βI, nearest/trace/legacy, I-STDP on/off, dt, seed）は出尽くしており、これ以上のスカラー調整では同じ壁に当たる。
- **統一仮説(高分解能解析より)**: 遷移点はナイフエッジで、EE暴走(σ>1)かEI暴走(σ<1)のどちらかに倒れて σ=1 を保持できない。
- **検証で判明した根本原因(重要)**: これは興奮性の復元力欠如(βE)ではなく、**escape noise の gain が急峻すぎること(b=4)**。
  - βE=1.05(復元力付与)は不発（過抑制σ<1のまま）。
  - **b=8mV(gainを浅く, 2倍速い重み成長を dσ/dw 半減で補償)で初めて σ=1 を安定保持**（ΔCr≈0, E-I均衡, 抑制非飽和, 全5seed, 12h）。遷移も h3→h8 と論文の~6hに接近。
  - ⇒「重み成長が2倍速い」問題が σ=1 不安定として顕現し、**gain(b)で補償可能**。72h保持の確認と b の微調整(4〜8)が次段階。

## 6. 著者への確認事項（再現に必要な未公開の実装詳細）

1. **I-STDP の恒常的安定化**: βI=1.15（正味抑圧的カーネル）で、なぜ高活動の遷移時に抑制が崩壊せず、
   安定臨界（branching≈1）を維持できるのか。実コードでの IE の収束機構・目標発火率の設定。
2. **学習率 A=0.02 の単位**: Stepp は重み・学習率を nS（絶対値）で定義。本論文は重みを無次元[0,1]に正規化(g_max=4.0)。
   A=0.02 は無次元重みに対する 1 ペアあたりの変化量という理解で正しいか（実効値・換算係数）。
3. **g_max=4.0 の扱い**: Table S1 の "Maximum **total** conductance of excitatory inputs" の意味。式 S9 は per-synapse 適用だが、
   実コードで fan-in/N による正規化はあるか（無いと我々の解釈と一致、あると我々は過結合）。
4. **ニューロンの離散化詳細**: escape 確率 S5 を v(t) と v(t+dt) のどちらで評価するか、スパイク判定とシナプス入力・
   リセット・不応期中の V の扱いの順序（Brian2 実装の update 順）。
5. **STDP の時刻基準**: S13/S14 の t_pre/t_post は生スパイク時刻か到着時刻か（遅延の扱い）。

## 7. 機構の定量的定式化: b と g_max が何を制御するか

本節は「b が σ に、g_max が E-I 比に効く」という主張を、**用語を先に厳密に定義**した上で命題として導く。
以降 proportionality は order-1 の定数（膜/シナプス核の積分など）を吸収した記法で、符号と支配変数の依存関係のみを主張する。

### 7.0 用語の定義

- **発火強度（escape-noise LIF）** `ρ(V) ≡ f_rest · exp((V − v_rest)/b)` [Hz]。膜電位 V の連続関数として発火確率密度を与える（決定論的閾値の代わり）。`f_rest`=静止発火率, `v_rest`=静止電位, **`b`=エスケープ幅（gainパラメータ, [mV]）**。
- **ニューロン利得（gain）** `G ≡ dρ/dV = ρ/b` [Hz/mV]。「1 mV の脱分極が発火率を何 Hz 増やすか」。定義から `G ∝ 1/b`。
- **1スパイク後コンダクタンス増分（式 S9）** `Δg = U · x · w · g_max`。`U`=放出確率, `x`=STP資源(下記), `w`=無次元重み∈[0,1], `g_max`=最大コンダクタンス係数。これが後ニューロンを `ΔV ≈ κ · Δg`（κ=膜・シナプス核の積分, [mV/コンダクタンス]）だけ動かす。
- **STP資源** `x ∈ [0,1]`（Tsodyks 抑圧変数）。発火で枯渇し `τ_rec`=150ms で回復。定常平均を `x̄` と書く。`x̄` は発火率の減少関数（高活動ほど枯渇）。
- **分岐比（branching ratio）** `σ ≡` 「1個の前スパイクが直接誘発する後スパイクの期待数」。線形化すれば **実効1ステップ伝播行列 J のスペクトル半径**。`σ<1`=劣臨界（活動は減衰）, `σ=1`=臨界（べき乗）, `σ>1`=超臨界（発散/同期）。
- **I-STDP の恒常性セットポイント** `ρ*`。Vogels-Sprekeler 型 I-STDP は、抑制性重みを調整して**後（興奮性）ニューロンの発火率を目標値 `ρ*` に引き込む**。`ρ*` は STDP カーネルの増強/抑圧の比（≈ βI と目標率パラメータ）で決まり、**`g_max` に依存しない定数**。
- **飽和（saturation）** 重みが上限 `w=W_max=1` に達し、可塑性が「それ以上動かせない」状態＝**アクチュエータの喪失**。

### 7.1 命題1: b はニューロン利得を介して σ の**安定性**を支配する

**(a) σ への依存.** 興奮性 j→i（ともに E）の実効1ステップ利得は
`J_ij = (ΔV per spike) × (発火率感受性) = [κ · U · x̄ · w_ij · g_max] × [dρ_i/dV] = κ U x̄ w_ij g_max · (ρ_i/b)`。
よって興奮性部分網のスペクトル半径 `σ ∝ (1/b) · g_max · x̄ · w̄_EE · (構造因子)`、したがって
**`∂σ/∂w ∝ g_max / b`**（1つの重みが動いたときの σ の感度）。

**(b) 安定性論法.** SoC は `dw/dt = F(σ)`（E-STDP が σ>1 側で w を増やし、STP枯渇＋I-STDP が減らす）という制御ループ。固定点 σ*=1 近傍で
`dσ/dt = (∂σ/∂w)(dw/dt)`。恒常性の復元は有限速度。**`∂σ/∂w (∝ g_max/b)` が大きいと、わずかな w のドリフトが大きな σ 逸脱を生み、遅い恒常性が追いつく前に σ=1 を通過**→暴走(σ>1)か過抑制(σ<1)へ倒れる（ナイフエッジ双安定）。`g_max/b` を下げる（b を上げる or g_max を下げる）と σ 逸脱が縮み、恒常性が追従して **σ=1 が安定吸引点**になる。本実装は重み成長が論文比~2倍速く `dw/dt` が過大 → b を2倍にして `∂σ/∂w` を半減し相殺、が b=8 成功の定量的根拠。

**(c) b の第2の効き（副作用）.** `ρ = f_rest·exp((V−v_rest)/b)` は V>v_rest で `∂ρ/∂b < 0`。**b を上げると活動発火率が全体的に下がり（f-I 曲線を f_rest 側へ圧縮）**、E-STDP の駆動が弱まって W_E が天井に届かなくなる。故に b は「純粋な σ ノブ」ではなく、**発達オーバーシュートのエンジンも弱める**。データ: b5→b8 で exc 3.9→1.2、Wei(mean) 0.43→0.50 止まり（天井未到達）＝オーバーシュートが弱い。

### 7.2 命題2: g_max は**恒常性の非対称性**を介して E-I 発火率比を支配する

**ナイーブな反論とその誤り.** `g_exc = U x W_E g_max ρ_E`, `g_inh = U x W_I g_max ρ_I` なら比で g_max が消える——**これは ρ_E, ρ_I が両方自由な場合のみ**。実際は2つの制約で対称性が破れる:

1. **I-STDP が ρ_E を ρ\*（g_max 非依存の定数）に固定**（サーモスタット）。→ 分母 ρ_E は g_max で動かない。
2. **E-STDP(βE=1.0, 復元力なし)が W_EI を天井 1.0 に固定**。→ E→I ドライブ `= U x̄ · W_EI · g_max · ρ_E = U x̄ · 1.0 · g_max · ρ* ∝ g_max`。

抑制性ニューロンはこのドライブで発火するので **`ρ_I ∝ g_max`**。さらに W_II が飽和して I 群の発火を抑え戻せない（I 群サーモスタットのアクチュエータ喪失）ため ρ_I は青天井。結論:
**`ρ_I/ρ_E = (∝ g_max)/(≈ρ* 一定) ∝ g_max`**。分子だけ g_max で動くのでキャンセルしない。
直感: I-STDP は「興奮性の温度計」。結合(g_max)を強めても ρ_E≈ρ* を保つが、そのために必要な抑制は**重みが天井**で頭打ち→増やせるのは**抑制スパイク数(ρ_I)**のみ→g_max の増分は丸ごと ρ_I に吸収され比が上がる。
データ(72h): gmax2.0→3.0 で exc≈1(一定), inh 2.3→4.1, I/E 2.4→3.3。

### 7.3 系: per-neuron 発火率分布の非対称性（E二極化・I収束）

同じ非対称性が個々のニューロン分布に現れる（gmax2.5, seed2, CV=std/mean）:

- **興奮性は「集団平均のみ」制御される**。I-STDP は平均 ρ_E を ρ* に保つが、個々の E は E-STDP で分化する自由がある（強い再帰経路 W_EE=天井 に入った少数が高発火、他は低発火）→ **平均~1 の周りに分布が広がる**。データ: EXC CV 6h:0.10 → 72h:**0.30**（範囲 0.46–1.67, 上位9%が>1.5Hz）＝**二極化**。
- **抑制性は「共通ドライブに隷属」**。I 群は E 集団全体を平均サンプルし、W_II 飽和で個別制御も失う → **同一の g_max 決定ドライブに収束**。データ: INH CV 72h:**0.04**（範囲 3.1–3.8）＝~3.4 に密収束。

（対照）b8 は EXC CV 0.09・INH 1.57(CV0.07) と両群タイト＝二極化も高inh収束も起きず、per-neuron でも非論文的。

### 7.4 g_max を興奮/抑制で非対称化した場合の予測（g_max_E ≠ g_max_I）

`g_max_E`（興奮性シナプス係数）と `g_max_I`（抑制性シナプス係数）を分離する。

- **σ は主に `g_max_E/b` が支配**。7.1(a) の J で興奮性利得は g_max_E に比例。抑制は E–I フィードバックループを通じてスペクトル半径を**下げる**（安定化）方向に効き、その強さは `g_max_I · W_IE` に依存。
- **現行の g_max(対称) 群は劣臨界(ΔCr≈−0.5)＝抑制が σ を 1 未満へ削りすぎ**。したがって **`g_max_I` を下げる（抑制を弱める）と σ が 1 へ上がる**——これは「抑制:興奮 = 1:4（抑制を 1/4）」の**方向としては正しい**。
- **ただし恒常性による部分打ち消しあり**: I-STDP は平均発火率を通じ net 抑制を復元しようとする。効くのは主に**動的(過渡)抑制の低下**で σ が上がる分（平均は恒常性が戻すが、σ=分岐は過渡量なので完全には戻らない）。
- **トレードオフと risk（要注意）**:
  - **絶対値が重要**。`g_max_E` を論文の 4.0 のままにすると `g_max_E/b=1.0`（高感度）で、弱い抑制では**暴走(σ>1)** に倒れる恐れ。→ `g_max_E` は中庸(~2.5, `g_max_E/b≈0.6`)に保つべき。
  - `g_max_I` を下げると同じ net 抑制に**より大きな W_I** が要り **W_I が一層飽和**（重みアクチュエータをさらに失う）→ σ=1 保持が rate 制御頼みになり不安定化しうる。
  - `ρ_I/ρ_E` は 7.2 より `g_max_I` を下げると I-STDP が ρ_I を上げて補償 → **E-I 比が論文の~3 を超過**しうる。
- **予測と推奨**: 「抑制を弱める」方向は劣臨界脱出に整合。**推奨スイープ: `g_max_E ∈ {2.5, 3.0}` × `g_max_I ∈ {g_max_E, g_max_E/2, g_max_E/4}`**。`g_max_I = g_max_E/2〜/4`（=2:1〜4:1）が ΔCr を −0.5 → 0 側へ寄せつつ gmax2.5 の良好な現象論を保つ候補。ただし **これは Table S1（g_max_exc=g_max_inh=4.0）からの逸脱**であり、「論文再現」ではなく「何が臨界を安定化するか」の探索である点に注意。要 supplementary 再確認。
