# パラメータ参考表

**どの文献の・どの測定量を・どう解釈して・モデルのどの量に落としたか**を記録する。


## 記号の凡例

このドキュメントの目的は「根拠がある値」と「根拠が弱い値」を分けること。
各値に、その根拠の強さを表す記号を 1 つ付ける。

| 記号 | 意味 |
|---|---|
| ✅ | 一次資料(論文本文)で数値と測定条件を確認済み |
| 🟡 | 出典名はあるが一次資料は未確認。孫引きの可能性があり |
| 🔶 | 文献値ではない。モデル都合・数値計算都合・規約で決めた値 |

なお、値そのものの根拠とは別に、**解釈や使い方に落とし穴がある**もの
(近似・スケール縮退・特定条件下でのみ有効 など)は各所の本文で注記する。

|---|---|---|
| 膜電位 | mV | |

| 膜抵抗 `R_m` | **MΩ** | |

## 5. 伝導遅延 (`delays.yaml: distance_based`)

`delay(d) = d / velocity + synaptic_delay`

| パラメータ | 値 | 根拠 | 状態 |
|---|---|---|---|
| `velocity` | 500 µm/ms | 無髄軸索伝導速度 (Swadlow 1994) | 🟡 🔶 一次未確認かつ実装都合の上限あり(下記) |
| `synaptic_delay` | 1.0 ms | 最小シナプス遅延 (Sabatini & Regehr 1996) | 🟡 |
| `max_delay` | 25.0 ms | **実装都合**(下記) | 🔶 |

🔶 **`max_delay = 25.0` は生物学的な値ではない。**
NetworkBuilder が遅延を uint8 ステップで持つため、`255 × dt = 25.5 ms` を超えると
**無言で折り返す**。そのキャップ。同じ理由で `velocity = 500` も、
実測中央値としてよりは「1500 µm の対角線で折り返さない」ための下限として効いている。
恒久対策(uint16 化など)は未実施。

## 6. 初期重み (`weights.yaml: lognormal_broad`)

対数正規分布を任意で `[w_min, w_max]` にクリップ。利用者は**目標平均 `mean`(線形スケール)**と
**広がり `sigma_ln`** を指定するだけでよく、下地正規分布の平均
`mu_ln = ln(mean) − sigma_ln²/2` は `LogNormalRandomWeight` が逆算する
(人間が手計算しない)。

| パラメータ | 値 | 根拠 | 状態 |
|---|---|---|---|
| `mean` | 0.5 | 線形スケールでの目標平均。`mu_ln` は自動逆算 | 🔶 `g_max` と縮退(§6.1) |
| `sigma_ln` | 1.0 | 分布の広がり(CV に対応)。スケール不変で唯一独立に決まる量(§6.1) | 🟡 σ の実値は一次未確認 |
| `w_min` / `w_max` | 0.0 / 1.0 | custom_Akita のハードバウンド = 規約 | 🔶 |

**根拠**: シナプス強度が対数正規分布に従うという**形状そのもの**は Song et al. 2005 (PLoS Biol) が
L5 錐体ペア数百組の記録から明示的に報告している ✅
("the distribution of synaptic connection strength ... can be fitted by a lognormal distribution")。
一方で採用値 `sigma_ln = 1.0` が Song の当てはめ σ に一致するかは一次未確認(🟡)。

### 6.1 スケール縮退 — どの量に文献値が意味を持つか

`w` は無次元のスケール係数であり、実効コンダクタンスは `x_release · w · g_max`。
`w` の絶対スケールと `g_max` は**完全に縮退**している(`w` を定数倍しても `g_max` を
逆数倍すれば同一動態)。したがって:

| 量 | `g_max` への依存 | 文献値が意味を持つか |
|---|---|---|
| `mean`(絶対) | 縮退(`g_max` に吸収される) | **持たない**。単独では無意味。振幅は `g_max`(§7.3)で較正する |
| `sigma_ln`(形状 / CV) | **不変** (`g_max·w` は `μ` をずらすが `σ` は不変) | **持つ**。これが Song 2005 の本当の中身。裾の重さは `g_max` によらず保たれる |
| `mean / w_max`(相対位置) | 不変 | **条件付きで持つ**。クリップ + STDP がある限り効く(下記) |

- `mean` を文献から決めようとするのは筋が悪い。同じ動態は `mean` を変えて `g_max` を
  逆スケールしても得られるため、**振幅較正は `g_max` 側に外出し**している(§7.3)。
- **意味を持つのは `sigma_ln`(スケール不変)** であり、`sigma_ln ≈ 1`(0.8〜1.2)が
  文献の実体。これが「これだけが独立して決められる量」である。
- ただし `custom_Akita` の STDP が `[0, Wmax]` のハード上限を持つため純粋な縮退は破れる。
  クリップは分布形状を絶対位置に依存させ(例: `mean=0.7` が clip 後 ≈0.507 に変形)、
  STDP で重みが `Wmax` へ成長する余地は **`mean / w_max` 比**で決まる。この比は `g_max` に
  吸収されないので、`w_max = 1` に固定した上での `mean` の相対位置には意味が残る。

**運用**: `w_max = 1` を規約で固定 → `sigma_ln` を文献で決める → 絶対振幅は `g_max` で較正、
という切り分けにしている。`mean` は `mean/w_max` 比を通じた初期条件のノブと見なす。

### 7.2 短期抑圧 (Tsodyks-Markram)

| パラメータ | 値 | 状態 |
|---|---|---|
| `U` (初回放出率) | 0.4 | 🟡 |
| `tau_rec` (回復時定数) | 150 ms | 🟡 |

**これは無視できない効果である** ✅。定常発火率 r に対する平均資源は
`<x> = 1/(1 + U·tau_rec·r)` で、時間平均放出量は

```
U·<x>·r = U·r / (1 + U·tau_rec·r)   →   1/tau_rec = 6.67 /s  (r → ∞)
```

と**飽和する**。すなわち短期抑圧自体が暴走に対する負のフィードバックとして働く。

| r [Hz] | `<x>` | x=1 と仮定した場合の過大評価 |
|---|---|---|
| 1 | 0.943 | 1.06× |
| 5 | 0.769 | 1.30× |
| 20 | 0.455 | 2.20× |
| 50 | 0.250 | **4.00×** |

近似: 多接触結合(実際は 1〜3 接触)を単一放出部位として扱っているため、
抑圧の効き方は実際より階段的になる。平均振幅には影響しない。

### 7.3 `g_max` — 単発 PSP からの較正 ✅

文献のPSP 振幅を基準に`g_max` を決定。

#### 興奮性 — 確定できる

| | 値 |
|---|---|
| 参照 | **EPSP 0.7 ± 0.6 mV**、静止電位付近、L2/3 錐体ペア (Feldmeyer et al. 2002 J Physiol, PMC2343144) |
| 解釈 | unitary EPSP は「1 個の前ニューロンの 1 スパイクが後細胞に生む脱分極の**総和**」。接触点が複数あればその合計。モデルのシナプス 1 本も同じ量(前ニューロン i → 後ニューロン j の 1 結合)を表すので、**接続レベルで対応が取れている**。K_E が接触点数ではなく前ニューロン数を数えていることが、この対応の前提 ✅ |
| 結果 | **`g_max_E` = 0.0086 µS (無次元 0.863)** |
| 精度 | 高い。駆動力が `0 − (−74) = 74 mV` と大きく正確に分かるため |

#### 抑制性 — 帯にしかならない

| | 値 |
|---|---|
| 参照 | **IPSP 0.2 – 3.5 mV**、静止電位 **−55〜−60 mV**、FS 介在細胞 → 錐体細胞、逆転電位 **−70〜−82 mV** (Thomson, West, Hahn & Deuchars 1996 J Physiol, PMC1160826) |
| なぜ違う電位で測るのか | GABA_A 逆転電位が静止電位のすぐ近くにあるため、静止電位では駆動力がほぼゼロで IPSP を測定できない。逆に EPSP は脱分極させると発火してしまう。**測定手技の都合であって生理学的な差ではない**。ゆえに比較すべき不変量は振幅ではなくコンダクタンス |
| 問題 | 駆動力 `V_hold − E_Cl` が **15〜27 mV** と小さく、かつ `E_Cl` 自体が記録条件(ピペット内液の Cl⁻ 濃度)依存の幅を持つ。分子(IPSP)も 0.2〜3.5 mV と広い |

`g_max_I` [µS](括弧内は `g_I/g_E`、g_E=0.0086)。
EPSP 較正と同一の過渡ピーク積分(R_m=100 MΩ, tau_m=30 ms, **抑制 tau_syn=4 ms**, U·W=0.2, E_inh=−80):

| IPSP \ 保持電位 | −50 mV | −55 mV | −60 mV |
|---|---|---|---|
| 0.2 mV | 0.0034 (0.4) | 0.0041 (0.5) | 0.0051 (0.6) |
| 1.0 mV | 0.0174 (2.0) | 0.0209 (2.4) | 0.0263 (3.1) |
| **2.0 mV** | 0.0354 (4.1) | **0.0429 (5.0)** | 0.0543 (6.3) |
| 3.5 mV | 0.0640 (7.4) | 0.0781 (9.1) | 0.0999 (11.6) |


**結論: 文献の IPSP 幅 (0.2–3.5 mV) と保持電位幅 (−50〜−60) で `g_I/g_E` は 0.4〜11 の帯。
生理的に妥当な 1–2 mV @ −55 に絞れば 2.4〜5.0。単発コンダクタンス直接測定
(Gupta 2000 / Xiang 2002、いずれも全文有料で値未抽出)があれば駆動力不定性を回避できる。**

なお、体細胞周囲に投射する FS 介在細胞と樹状突起スパインに投射する興奮性入力では
樹状突起フィルタリングの受け方が異なるが、**点ニューロンモデルでは発火に効くのは
体細胞での効果**なので、体細胞記録値をそのまま使うのは整合的である ✅。
